# Videos Folder

Place your video files in this folder.

## Recommended Structure

- `brand-video.mp4` - Your 10-second brand video
- `exercise-of-the-week/` - Folder for exercise videos
- `workout-of-the-week/` - Folder for workout videos
- `promotional/` - Folder for promotional videos

## Supported Video Formats

- MP4 (recommended)
- WebM
- OGG

## File Naming

Use descriptive, lowercase filenames with hyphens:

- ✅ `brand-video.mp4`
- ✅ `push-up-tutorial.mp4`
- ❌ `Brand Video.mp4` (spaces)
- ❌ `brand_video.mp4` (underscores)

## Thumbnails

You can optionally add thumbnail images for your videos. Place them in the same folder with the same name but `.jpg` or `.png` extension:

- `brand-video.mp4` → `brand-video.jpg` (thumbnail)
